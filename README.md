# Syllabus Generator

A web-based tool for generating course syllabi from Development Guide markdown files.

## Features

- **Upload Course Files**: Drag and drop or select markdown (.md) Development Guide files
- **Automatic Extraction**: Extracts course information, outcomes, grading, and unit learning outcomes
- **Syllabus Generation**: Creates formatted syllabus documents with:
  - Course code and title
  - "What Will I Learn?" section
  - "How Will I Complete This Course?" section with formatted activity types
  - Course Outcomes from CLA mapping table
  - Grading and Assessment table
  - Unit Learning Outcomes organized by unit
- **Multiple Export Options**:
  - Download as Markdown (.md)
  - View in new tab (can print/save as PDF)

## Usage

1. Open `index.html` in a web browser
2. Upload one or more Development Guide markdown files
3. Click "Process Files" to generate syllabi
4. Use the download buttons to get your syllabus in desired format

## File Structure

- `index.html` - Main application (single-file HTML/CSS/JS application)

## Development

This is a standalone HTML application with no build process required. Simply edit `index.html` and refresh the browser to see changes.
